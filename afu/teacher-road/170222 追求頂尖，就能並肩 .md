# 追求頂尖，就能並肩 

<p><a href="http://4.bp.blogspot.com/-c7dWEDgpJ_M/WKz89kgf1-I/AAAAAAAEY5c/qlTHPqU3Xv4QHgofXrKUzPpGf7vmMsT_gCK4B/s1600/%25E5%258D%2583%25E8%2590%25AC%25E8%25AC%259B%25E5%25B8%25AB%25E7%259A%258450%25E5%25A0%2582%25E8%25AA%25AA%25E8%25A9%25B1%25E8%25AA%25B2-cover%25E5%25AE%258C%25E7%25A8%25BF-01%2B%25E5%25B0%2581%25E9%259D%25A2%2B%25E5%25B0%258F.jpg"><img border="0" height="260" src="https://4.bp.blogspot.com/-c7dWEDgpJ_M/WKz89kgf1-I/AAAAAAAEY5c/qlTHPqU3Xv4QHgofXrKUzPpGf7vmMsT_gCK4B/s640/%25E5%258D%2583%25E8%2590%25AC%25E8%25AC%259B%25E5%25B8%25AB%25E7%259A%258450%25E5%25A0%2582%25E8%25AA%25AA%25E8%25A9%25B1%25E8%25AA%25B2-cover%25E5%25AE%258C%25E7%25A8%25BF-01%2B%25E5%25B0%2581%25E9%259D%25A2%2B%25E5%25B0%258F.jpg" width="640"/></a></p>
<p>也許大家可能不曉得，在《千萬講師的50堂說話課》內文完稿前，我跟憲哥並沒有看過彼此寫的文章。我們只是一開始跟編輯開了幾次會，確定了大致寫作的方向，然後就開始動手寫了。等到真正看到彼此的文章，已經是最後在寫相互評論＆回饋的階段了。<br/>
<a name="more"></a><br/>
<a href="http://1.bp.blogspot.com/-IA-qdihwOGQ/WK0BO0iWtTI/AAAAAAAEY7U/Y7In3nbKrWQXDKGGa6avUT6Gq3OJ1ml7wCK4B/s1600/%25E8%259E%25A2%25E5%25B9%2595%25E5%25BF%25AB%25E7%2585%25A7%2B2017-02-22%2B%25E4%25B8%258A%25E5%258D%258810.38.07.jpg"><img border="0" height="454" src="https://1.bp.blogspot.com/-IA-qdihwOGQ/WK0BO0iWtTI/AAAAAAAEY7U/Y7In3nbKrWQXDKGGa6avUT6Gq3OJ1ml7wCK4B/s640/%25E8%259E%25A2%25E5%25B9%2595%25E5%25BF%25AB%25E7%2585%25A7%2B2017-02-22%2B%25E4%25B8%258A%25E5%258D%258810.38.07.jpg" width="640"/></a></p>
<p><i>（2016 寫出影響力，聆聽何飛鵬社長「論寫作」，何社長是我們的大貴人！）</i></p>
<p>從大家的回饋中，最終完成的書，整體性跟易讀性應該還不錯。當然這有很大的一部份是編輯大人的功勞。不過換個角度來看，你可以想像一下，如果你跟另外一個朋友，針對一個大主題共同寫作。在寫了 20～30 篇文章之後才匯整起來。你覺得這個是簡單？還是困難？會不會很大的風格差異？會不會有文筆的差異？你不會擔心 … 誰寫的比較好？誰寫的比較差？放在一起比較 … 會不會馬上就顯現出來了？</p>
<p><a href="http://3.bp.blogspot.com/-Wm8B5fi9ido/WKz9JpuCocI/AAAAAAAEY5k/-Am6Nivy2TwbvWMRKT1qT25wqnl4c1yrQCK4B/s1600/%25E8%259E%25A2%25E5%25B9%2595%25E5%25BF%25AB%25E7%2585%25A7%2B2017-02-22%2B%25E4%25B8%258A%25E5%258D%258810.33.54.jpg"><img border="0" height="456" src="https://3.bp.blogspot.com/-Wm8B5fi9ido/WKz9JpuCocI/AAAAAAAEY5k/-Am6Nivy2TwbvWMRKT1qT25wqnl4c1yrQCK4B/s640/%25E8%259E%25A2%25E5%25B9%2595%25E5%25BF%25AB%25E7%2585%25A7%2B2017-02-22%2B%25E4%25B8%258A%25E5%258D%258810.33.54.jpg" width="640"/></a></p>
<p><i>（2016.08.05 編輯會議，確認新書內容方向）</i></p>
<p>如果真的想像一下，你才會知道，這樣合作的過程中間，有許多不可控制的因素。重點是：我們倆個在過程中，總是彼此信任，從沒有擔心過這些問題。因為我知道（我只能代表我的觀點）：憲哥的文章一定會寫的極好，我只要用心寫，更好的方向前進，他一定會在那裡等著我。</p>
<p><a href="http://4.bp.blogspot.com/-APxtyhghPps/WKz9WwWyZ0I/AAAAAAAEY5w/m2lGMKsLxmUliIiKfmvz_1dvp3e97I1agCK4B/s1600/%25E8%259E%25A2%25E5%25B9%2595%25E5%25BF%25AB%25E7%2585%25A7%2B2017-02-22%2B%25E4%25B8%258A%25E5%258D%258810.34.24.jpg"><img border="0" height="456" src="https://4.bp.blogspot.com/-APxtyhghPps/WKz9WwWyZ0I/AAAAAAAEY5w/m2lGMKsLxmUliIiKfmvz_1dvp3e97I1agCK4B/s640/%25E8%259E%25A2%25E5%25B9%2595%25E5%25BF%25AB%25E7%2585%25A7%2B2017-02-22%2B%25E4%25B8%258A%25E5%258D%258810.34.24.jpg" width="640"/></a></p>
<p><i>（2016.08.05，專業簡報力演練，有注意到2位即將登上 TED x Taipei 的講者嗎？）</i></p>
<p>同樣的事情也出現在憲福講私塾的教學過程，有一段是我們要談企業內訓的強力開場。我自己當然有一套標準的強力開場流程，但是那時我還不大知道憲哥在企業內訓時是怎麼做的。因此我請他錄一段他在企業內訓進行開場的影片。結果看到影片後，我驚呆了！</p>
<p>「這根本是一模一樣的開場方式啊！」</p>
<p><a href="http://1.bp.blogspot.com/-zpMGYdbaT64/WKz-RVO1enI/AAAAAAAEY6E/Hj9CjYIo16IEPeu7O4mzXvjRe5dH6eK-wCK4B/s1600/%25E8%259E%25A2%25E5%25B9%2595%25E5%25BF%25AB%25E7%2585%25A7%2B2017-02-22%2B%25E4%25B8%258A%25E5%258D%258810.56.59.jpg"><img border="0" height="428" src="https://1.bp.blogspot.com/-zpMGYdbaT64/WKz-RVO1enI/AAAAAAAEY6E/Hj9CjYIo16IEPeu7O4mzXvjRe5dH6eK-wCK4B/s640/%25E8%259E%25A2%25E5%25B9%2595%25E5%25BF%25AB%25E7%2585%25A7%2B2017-02-22%2B%25E4%25B8%258A%25E5%258D%258810.56.59.jpg" width="640"/></a></p>
<p><i>（2017.02.11，憲福講私塾 4，一個本來停開，後來沒有招生就額滿的課程）</i></p>
<p>我們事前沒套好，彼此沒先溝通。但在這幾年的企業內訓經驗中，獨立摸索出自己覺得有效的教學方法。幾年之後開始合作，彼此對一下，結果發現雖然彼此摸索的路徑不大相同，但最終得到的結果卻是非常相同。</p>
<p><a href="http://4.bp.blogspot.com/-4OI8d7b3C0Q/WKz_ZtmnGjI/AAAAAAAEY6c/oljfNIcAufAeahM1wYMMIL6wS-RqGq9WACK4B/s1600/%25E8%259E%25A2%25E5%25B9%2595%25E5%25BF%25AB%25E7%2585%25A7%2B2017-02-22%2B%25E4%25B8%258A%25E5%258D%258811.02.08.jpg"><img border="0" height="240" src="https://4.bp.blogspot.com/-4OI8d7b3C0Q/WKz_ZtmnGjI/AAAAAAAEY6c/oljfNIcAufAeahM1wYMMIL6wS-RqGq9WACK4B/s640/%25E8%259E%25A2%25E5%25B9%2595%25E5%25BF%25AB%25E7%2585%25A7%2B2017-02-22%2B%25E4%25B8%258A%25E5%258D%258811.02.08.jpg" width="640"/></a></p>
<p><i>（千萬講師的50堂說話課，封面設計樣稿。最終我們選擇人少的顏色 …）</i></p>
<p>在個性上，我跟憲哥其實是很不一樣的兩個人（憲哥文章：<a href="http://lewis2fly.pixnet.net/blog/post/61050667-%E4%B8%8D%E4%B8%80%E6%A8%A3%EF%BC%8C%E5%8D%BB%E5%8F%88%E5%BE%88%E4%B8%80%E8%87%B4" rel="noopener" target="_blank">不一樣，卻又很一致</a>）。他大概可以一次處理 100 件事，我只能一件一件事情慢慢來。他在台上感染力十足，我習慣透過精細的教學法操作來達到我遇期效果。他的課程教大家說出影響力，不帶武器（投影片）就可以上台，強調的內功（說話的能力）。我的課程教大家專簡報力，把武器磨利（投影片），一樣可以用簡報（外功）達成目標。這麼不一樣的兩個人、不一樣的個性、不一樣的課程。卻可以目標一致、方向相同、合作許多很棒的課程（我們都相互擔任對方課程的教練＆評審），也培養出一群 … 一樣都很優秀的學習夥伴們。</p>
<p><a href="http://3.bp.blogspot.com/-q5mSr1Qunws/WKz_rcDvstI/AAAAAAAEY6k/__lsK3uMl14uyrlLyVZV0LYDARD8uGV3gCK4B/s1600/%25E8%259E%25A2%25E5%25B9%2595%25E5%25BF%25AB%25E7%2585%25A7%2B2017-02-22%2B%25E4%25B8%258A%25E5%258D%258810.46.43.jpg"><img border="0" height="424" src="https://3.bp.blogspot.com/-q5mSr1Qunws/WKz_rcDvstI/AAAAAAAEY6k/__lsK3uMl14uyrlLyVZV0LYDARD8uGV3gCK4B/s640/%25E8%259E%25A2%25E5%25B9%2595%25E5%25BF%25AB%25E7%2585%25A7%2B2017-02-22%2B%25E4%25B8%258A%25E5%258D%258810.46.43.jpg" width="640"/></a></p>
<p><i>（本書的重量級推薦者之一：葉丙成老師，一起參與說出影響力與專業簡報力的課程講評）</i></p>
<p>這麼不一樣，卻又很一致。為什麼？</p>
<p>我的看法是：只要共同追求頂尖，最後就能一起並肩！</p>
<p>其實這就好像在登山的過程一樣，往山頂的路也許有 N 條路線，在過程中也許大家不一定會遇的到。但是不管走那一條路，越接近山頂，大家的距離就越近，如果目標是登頂，不管走的是那一條路，最終一定會在山頂相會。</p>
<p><a href="http://1.bp.blogspot.com/-P4yNXGjdl-4/WK0AAV5ku3I/AAAAAAAEY6s/zAe8s6wYy8AVgJ56czlJcFb6I-z2SusTQCK4B/s1600/%25E8%259E%25A2%25E5%25B9%2595%25E5%25BF%25AB%25E7%2585%25A7%2B2017-02-22%2B%25E4%25B8%258A%25E5%258D%258810.49.34.jpg"><img border="0" height="478" src="https://1.bp.blogspot.com/-P4yNXGjdl-4/WK0AAV5ku3I/AAAAAAAEY6s/zAe8s6wYy8AVgJ56czlJcFb6I-z2SusTQCK4B/s640/%25E8%259E%25A2%25E5%25B9%2595%25E5%25BF%25AB%25E7%2585%25A7%2B2017-02-22%2B%25E4%25B8%258A%25E5%258D%258810.49.34.jpg" width="640"/></a></p>
<p><i>（事前完全沒有約好，卻在左營高鐵站相遇）</i></p>
<p>當然換一個角度來說，要能在山頂相會。至少目標要是一致的，也就是大家想要登上的山頭是同一顆。如果目標不同，或者是價值觀不同，也許彼此登上巔峰，但卻各自站在不同的山頭。這個並沒有對不對的問題，就只是漸行漸遠而已啊！</p>
<p>很開心，在登山的過程中，本來以為是孤獨前行，後來慢慢發現，有很多大神＆好兄弟（像Adam 哥、憲哥、MJ），已經走在不遠的前方等著我。有時大家往上選擇不同的路走，但是最後卻又能交會在一起。更開心的是：這幾年也陸續透過課程＆出書的機會，開始影響一群學習夥伴，一起朝向心中的山頂前進。這真的是一段很開心的過程。</p>

<p><i>（2017.02，公司聯合旅遊）</i></p>
<p>這幾年還是要謝謝憲哥，總是走的比我快，在前面引領我們發展的方向。還好我腳力也還行，雖然走不快，但要跟上還是有能力的。（憲哥文章：<a href="http://lewis2fly.pixnet.net/blog/post/65029039-%E5%85%A9%E5%B9%B4%E5%BE%8C%E7%9A%84%E5%8D%83%E8%90%AC%E8%AC%9B%E5%B8%AB%EF%BC%8C%E4%BA%94%E5%8D%81%E5%A0%82%E7%9A%84%E7%84%A1%E5%83%B9%E5%AD%B8%E7%BF%92" rel="noopener" target="_blank">兩年後的千萬講師，五十堂的無價學習</a>）</p>
<p><a href="http://1.bp.blogspot.com/-m-ZclgRCh4M/WK0BXAx9BwI/AAAAAAAEY7c/BZuVeG2DU30aE3h3lf19_og2Nl3y1cy_gCK4B/s1600/%25E8%259E%25A2%25E5%25B9%2595%25E5%25BF%25AB%25E7%2585%25A7%2B2017-02-22%2B%25E4%25B8%258A%25E5%258D%258810.47.41.jpg"><img border="0" height="480" src="https://1.bp.blogspot.com/-m-ZclgRCh4M/WK0BXAx9BwI/AAAAAAAEY7c/BZuVeG2DU30aE3h3lf19_og2Nl3y1cy_gCK4B/s640/%25E8%259E%25A2%25E5%25B9%2595%25E5%25BF%25AB%25E7%2585%25A7%2B2017-02-22%2B%25E4%25B8%258A%25E5%258D%258810.47.41.jpg" width="640"/></a></p>
<p><i>（2016.12，說出生命力評審，為台上講者的生命故事感動中）</i></p>
<p>當然，一山還有一山高，我們登上我們山頂後，也許還會看到更高的山頭。這時我們也不會過於留戀，會再次走下山，然後往更高的山頭邁進。只要共同追求頂尖，最後還是能再次並肩。一起加油，一起朝向更高的地方邁進的！</p>
<p>在往山頂的路上，我們等你！一起來吧！</p>
