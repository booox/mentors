# 葉丙成老師推薦簡報書 :《上台的技術》

<p>出書幾個月之後，前晚深夜，忽然看到翻轉教學神人，台大 Mooc 執行長 : <a href="http://pcyeh.blog.ntu.edu.tw/">葉丙成老師</a>，在 <a href="https://www.facebook.com/pcyeh.NTU/posts/1121132294579134?pnref=story">FB 上 PO 文推薦</a><a href="http://www.books.com.tw/products/0010657348">《上台的技術》</a>！他說：</p>
<p>「坊間諸多關於簡報技巧的書，福哥（王永福）的書「<a href="http://www.books.com.tw/products/0010657348"><b>上台的技術</b></a>」<b>是我最推薦的一本</b>。這也是為什麼我幫這本書寫推薦序。真的有興趣學好簡報的朋友，可以好好研讀此書，功力必然大增」</p>
<p>我真是高興萬分，快從椅子上掉下來了！<br/><a name="more"></a></p>
<p><a href="http://2.bp.blogspot.com/-YEzpkQVHBJU/VhKjqTcwdpI/AAAAAAAALTQ/RRY4dlLNBsw/s1600/image_thumb_99343e53ccdcf6a5a922d4b91799d5f4.png" style="margin-left: 1em; margin-right: 1em; text-align: center;"><img border="0" height="478" src="https://2.bp.blogspot.com/-YEzpkQVHBJU/VhKjqTcwdpI/AAAAAAAALTQ/RRY4dlLNBsw/s640/image_thumb_99343e53ccdcf6a5a922d4b91799d5f4.png" width="640"/></a></p>
<p>其實跟葉老師的認識，是一件沒有刻意的緣份。幾年前朋友 Aggie 問我能不能幫她一個忙，去台大電機系教一門專案簡報課。我平常僅在公司進行內訓，基本上沒有在校園裡教學。但基於幫朋友的忙，我就應允了這個課程。從此連續3年，每年都有一天去台大電機教簡報。一直到 2014 年教完。我就沒教了。</p>
<p><a href="http://2.bp.blogspot.com/-roU3C_q0YFY/VhKjq1keTVI/AAAAAAAALTc/lOQENfwT9cg/s1600/image_thumb_ab6afb22074d4924aa8321e59f4dfdd5.png" style="margin-left: 1em; margin-right: 1em; text-align: center;"><img border="0" height="294" src="https://2.bp.blogspot.com/-roU3C_q0YFY/VhKjq1keTVI/AAAAAAAALTc/lOQENfwT9cg/s640/image_thumb_ab6afb22074d4924aa8321e59f4dfdd5.png" width="640"/></a></p>
<p>這幾年我也知道，葉丙成老師除了推動翻轉教學外，也在台大開了簡報技巧的課程，只是校園那麼大，我們也不太可能遇的到。但是還好有 FB，之後我主動加了葉老師為 FB 好友，只是我們還是一直沒有見過面。僅止於 FB 上的相互關注！</p>
<p>有一天深夜，葉老師突然私訊給我，然後就換成電話聊起來了，一聊就一小時，雖然沒見過，但是2個從沒見過面的成年宅，對彼此都蠻熟悉的。原來他的愛徒，上過我在台大二次的簡報課，也跟他推薦我的簡報教學。所以我們直接談到重點，交換對簡報及教學上的一些不同意見。我對葉老師開朗、直爽印象很深刻，我們 Tone 也蠻合的，那天電話聊很開心！</p>
<p><a href="http://3.bp.blogspot.com/-d4MrSqb3UMU/VhKjpqpcXdI/AAAAAAAALTI/B98bMEBw8fU/s1600/image_thumb_21155aa5cea8210587293389c3ed3d50.png" style="margin-left: 1em; margin-right: 1em; text-align: center;"><img border="0" height="476" src="https://3.bp.blogspot.com/-d4MrSqb3UMU/VhKjpqpcXdI/AAAAAAAALTI/B98bMEBw8fU/s640/image_thumb_21155aa5cea8210587293389c3ed3d50.png" width="640"/></a></p>
<p>後來葉老師邀我擔任<a href="http://udn.com/news/story/6899/588959-%E5%8F%B0%E5%A4%A7%E7%9B%83%E7%B0%A1%E5%A0%B1%E5%A4%A7%E8%B3%BD-%E9%87%91%E7%8D%8E%E4%BD%9C%E5%93%81%E9%A9%9A%E8%89%B7%E8%A9%95%E5%AF%A9">台大簡報大賽評審</a>，我也非常榮幸的答應了。那一天剛好是新書出版第一天，我跟編輯及行銷也親自把書帶到現場，送給葉老師及同學們，也很感謝葉老師在新書上具名推薦。當天是我們第一次見面，我那天感覺不錯，在講評上也蠻有 fu 的，看的出來葉老師也蠻開心的。跟二位評審（聯合線上李總、微軟康總）一起當評審也是很棒的經驗。</p>
<p><a href="http://4.bp.blogspot.com/-m5031oplgZg/VhKjpgra86I/AAAAAAAALTM/IE36q5ZvGi4/s1600/image_thumb_09ef02b0a053c737d2ae5fa5019e1342.png" style="margin-left: 1em; margin-right: 1em; text-align: center;"><img border="0" height="480" src="https://4.bp.blogspot.com/-m5031oplgZg/VhKjpgra86I/AAAAAAAALTM/IE36q5ZvGi4/s640/image_thumb_09ef02b0a053c737d2ae5fa5019e1342.png" width="640"/></a></p>
<p>後來我跟葉老師再約一天，一起在台大吃早餐。雖然一整個早上亂聊 … 但是很開心有機會近距離了解葉老師教學理念的形成，以及未來的一些目標。其實除了大家知道的神表現（例如拿到<a href="http://www.appledaily.com.tw/appledaily/article/headline/20141211/36259860/">全球第一教學創新大獎</a>，或是<a href="https://www.google.com.tw/search?num=100&amp;newwindow=1&amp;es_sm=119&amp;biw=1024&amp;bih=561&amp;q=%E8%91%89%E4%B8%99%E6%88%90+%E9%9B%9C%E8%AA%8C&amp;oq=%E8%91%89%E4%B8%99%E6%88%90+%E9%9B%9C%E8%AA%8C&amp;gs_l=serp.3...613085.615369.0.615741.13.10.1.0.0.0.225.922.2j4j1.7.0.msedr...0...1c.1j4.64.serp..10.3.280.C6Nqun5vUNg">一卡車的媒體報導</a>，以及<a href="https://www.google.com.tw/search?q=%E8%91%89%E4%B8%99%E6%88%90+%E7%BF%BB%E8%BD%89%E6%95%99%E5%AD%B8+%E6%BC%94%E8%AC%9B&amp;num=100&amp;newwindow=1&amp;es_sm=119&amp;biw=1024&amp;bih=561&amp;source=lnms&amp;tbm=vid&amp;sa=X&amp;ei=TFMnVcrvG42D8gXTvoHQDA&amp;ved=0CAkQ_AUoAg">環台環世界的翻轉教學分享</a>，還有在 <a href="https://zh-tw.coursera.org/course/prob">Coursera 著名的機率課</a>）。我對葉老師最敬佩的是：他永遠會把光環跟團隊共享，重情重義！看他在 <a href="https://www.youtube.com/watch?v=7htkDYcZHUg">2013 在 TED x Taipei 分享</a>時，最後邀請團隊的成員上台，這才是令我最敬佩及感動的地方。</p>

<p>在他趕完新書之後（預定5月出版，必買！），還不忘再推《上台的技術》一把，這份情義 … 我收到了！感謝啊！<b>以下為葉老師的推文及登載在新書上的推薦文，原文照登</b>：</p>
<p><a href="http://3.bp.blogspot.com/-TsAR4KETaHU/VhKjoWlU8BI/AAAAAAAALTA/HiRoM7Bk8L4/s1600/image_thumb_5e89bafeea0f67315d39f4bd1196f598.png" style="margin-left: 1em; margin-right: 1em; text-align: center;"><img border="0" height="288" src="https://3.bp.blogspot.com/-TsAR4KETaHU/VhKjoWlU8BI/AAAAAAAALTA/HiRoM7Bk8L4/s640/image_thumb_5e89bafeea0f67315d39f4bd1196f598.png" width="640"/></a></p>
<p>最近常有出版社希望找我寫本談簡報技術的書，不過我因為事務太多無法寫書而一直推辭。坊間諸多關於簡報技巧的書，福哥（王永福）的書「上台的技術」是我最推薦的一本。這也是為什麼我幫這本書寫推薦序。真的有興趣學好簡報的朋友，可以好好研讀此書，功力必然大增。</p>
<p>以下是我寫的推薦序，提供大家參考：</p>
<p>———————</p>
<p>《一本教你享受上台的好書！》</p>
<p>我跟福哥，從沒見過面，卻對彼此很熟悉。</p>
<p>五年前，我創立了台大第一門以十八週正規學期時間講授的「簡報製作與表達」課程。每年有四百多位台大同學想修。但為了讓每位同學都能講到話，我們每班只能收四十五位同學。最後得以修課的同學，往往是在修課申請書中，讓我們看到最有決心想改變自己的同學。他們也在這個很特別的課程中，跨出了自己的舒適圈，真正的改變了自己。</p>
<p>之所以初識福哥，是來自學生的介紹。從我這群極有熱忱提昇自己簡報能力的愛徒們口中，我常聽到某個名字：「老師！有位福哥講的簡報課也是超讚的！」後來兩個重度臉書使用者因緣際會互加臉書。拜臉書之賜，我們得以看到彼此的每日動態，相互關注回應。結果，兩個沒見過面的人，卻變成好友～</p>
<p>福哥最讓我敬佩的是，他雖然已是高階簡報企業課程中最廣為人知的簡報大師，但他還是時時努力讓自己更進步。大家很難想像，像福哥這麼忙的人，居然會一大早從台中拼到台北去上整整兩個全天的課程、寫了滿滿的筆記。福哥這種為了教好自己學生而不斷自我提升的毅力，真的非常讓我很感動。</p>
<p>另外，福哥最讓我有共鳴的，是他對完美的絕對堅持。在臉書上，不時看到福哥為了讓本書能更完美呈現，不斷的修改、修改、修改。福哥把追求完美的精神，透過這本書傳遞給所有的讀者。這種追求完美呈現自己的極致思維，正是一個簡報能成功的最關鍵要素，而福哥完全的讓大家感受到了！</p>
<p>有幸在本書付梓之前，得以先拜讀全書。福哥在書中提到的許多觀念跟範例，恰恰是我在課堂上不厭其煩跟學生們強調的點。書一看完，頓有知音知己之感！沒有機會修到台大簡報課或福哥簡報課的朋友們，我相信絕對可以從這本書中得到很多重要的簡報觀念。</p>
<p>但要提醒各位讀友，知道觀念是一回事，真正能用出來又是另一回事。讀完「<a href="http://www.books.com.tw/products/0010657348">上台的技術</a>」這本好書，大家還是需要不斷的「上台」，才能有機會鍛鍊出屬於自己的真正「技術」。</p>
<p>願本書能幫助各位找到上台的自信！</p>

<p>寫完這篇文章後，我有幾個感想：</p>
<p>1.人脈經常不是刻意培養的，而是「作好事，把事作好」後自然形成的。如果沒有當初答應朋友，去台大電機教簡報課。然後不論鐘點費多少，還是努力把課教好 …。又怎麼會有後續這些緣份呢？您說這是刻意培養？或是非刻意呢？</p>
<p>2.永遠記得跟團隊分享光榮，永遠記得誰在過程中幫您一把。有時獲得大尊敬的，不是工作表現上的光環，而是您如何真心誠意的作人！</p>
